Kesimpulan

Berdasarkan analisis data sekuens Hepatitis B, diperoleh nilai GC Content yang relatif seragam pada seluruh sekuens. Sekuens OZ221990.1 memiliki nilai GC Content tertinggi yaitu 48.53%, diikuti AF090842.1 sebesar 48.37% dan NC_075114.1 sebesar 48.28%. Program berhasil membaca file FASTA, menghitung frekuensi nukleotida menggunakan Dictionary, mengurutkan data berdasarkan GC Content, menampilkan 3 sekuens terbaik, membuat visualisasi grafik, serta menyimpan hasil analisis ke dalam file CSV.
